Lively

About This Project

Lively is a full stack application for students to sublet their apartments to other students.

Navigating housing in NYC as a student is complicated, especially when you're not planning to be in the city the whole year. Finding someone to sublet to can be an arduous, complicated process. Lively is here to change that. As an app that's from students and for students, Lively promises a secure, trustworthy platform for students to list and search for apartments to sublet.

Lively is being built using Flask for the backend, React for the frontend, and MongoDB for the database.

Currently, Lively is in the very early stages of development. Some features I'm going to integrate include two types of user accounts for renters and listers, Oauth2 authentication, an apartment search function that can be customized to user preferences, a save apartment feature, and a chat for users to interact with each other.
