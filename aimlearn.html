AIMLearn

About This Project

AIMLearn is a project that I built during HackPrinceton that employs sentiment analysis to improve generative AI output. If users' facial expressions cross an intensity threshold, the AI will adjust how it delivers its response. Emotions displayed are broadly categorized as either positive (e.g. confidence) negative (e.g. confusion), or neutral (e.g. focused).

The vision for AIMLearn was to develop an educational tool that can aid both students and educators by utilizing AI & ML tools. Students use the query function to receive more personalized feedback by using continuous sentiment data. Educators access student queries and see points where students display strong reactions. This allows them to grasp how students are comprehending the material and how their lesson plans can be improved.

The tech stack I worked with involved using Next JS for backend and React JS for frontend. I also integrated OpenAI's GPT-3.5 model and Hume AI's Expression Measurement API.

In the future I plan to flesh out the account system and integrate Google APIs using OAuth2 authentication so it can have more real-world applicability with student and educator emails. I'm also planning to connect a MySQL database to store user data.
