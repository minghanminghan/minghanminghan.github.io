Game of Life

About This Project

This project is a recreation of Conway's Game of Life.

The Game of Life is a cellular automaton, meaning the game is represented by a board cells in one of two states, either 'dead' or 'alive'. It is also a zero-player game, meaning the outcome of each game is predetermined by the board's initial state.

Cells follow 2 simple rules:
(1) Alive cells with 2 or 3 alive neighbors (adjacent cells) stay alive
(2) Dead cells with exactly 3 alive neighrbors come to life

My rendition of the Game of Life was built with Python, and the video shows a GUI built with Pygame. In the video, the initial seed is randomized, and many patterns can be identified. These include static square blocks and ring-shaped beehives, as well as rotating 3-block blinkers, and less predictable bundles of cells.

I'm working on hosting the game on an static website so people can access it easier and play the game. I'm also trying to integrate the NumPy library to optimize the code further.
